## 📝 Sobre o Projeto

Este projeto consiste numa aplicação web desenvolvida para a disciplina de **Programação de Sítios Internet**. A aplicação consome a **Superhero-api** (versão disponível no GitHub), permitindo a consulta de diversos heróis e vilões.

Para a construção do projeto, foi utilizado **JavaScript** para a criação dinâmica de elementos HTML e **CSS** para a estilização da interface. A aplicação conta com tratamento de erros eficiente: caso um personagem não seja localizado, o sistema retorna uma mensagem informativa de "personagem não encontrado".

---

## 🎯 Objetivo

Criar uma aplicação web utilizando **JavaScript puro (Vanilla JS)** para consumir dados de uma **API pública**, exibindo os resultados de forma dinâmica numa interface amigável e intuitiva.

---

## 💡 Funcionalidades

- **Campo de busca:** Pesquisa por nome do personagem.
- **Consumo de dados:** Integração via API com a função `fetch()`.
- **Exibição Controlada:** Os resultados foram limitados a no máximo **3 cards por busca**, garantindo um layout limpo e organizado.
- **Manipulação Dinâmica:** Criação de elementos do DOM em tempo real.
- **Tratamento de Erros:** Feedback para buscas sem sucesso.
- **Responsividade:** Interface adaptável para diferentes dispositivos.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação da página.
- **CSS3:** Estilização, layout e efeitos de interatividade (*hover*).
- **JavaScript (Vanilla JS):** Lógica de consumo, filtragem e renderização.
- **API Utilizada:** [SuperHero API (akabab)](https://akabab.github.io/superhero-api/api/)
